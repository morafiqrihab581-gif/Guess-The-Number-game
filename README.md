# 🎯 Guess The Number Game

A fun and interactive number guessing game built with HTML, CSS, and JavaScript.

Players must guess a randomly generated number between 1 and 100 within a limited number of attempts. The project includes input validation, guess history tracking, restart functionality, keyboard support, and a responsive modern interface.

---

## 🎮 Game Rules

- Guess a number between **1 and 100**
- You have **10 attempts**
- The game provides hints:
  - Too high
  - Too low
- The game ends when:
  - You guess correctly
  - You run out of attempts
